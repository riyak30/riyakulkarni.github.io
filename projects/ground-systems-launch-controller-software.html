Ground Systems Launch Controller Software (SARP)

Built Python-based launch controller software that increased ground-system flexibility and mid-flight transmission modeling, while validating telemetry across Raspberry Pi, IGS, and OpenMCT for reliable mission operations.

Highlights

  • Developed Python software for launch controllers, adding flexibility to ground systems and mid-flight transmission modeling.
  • Conducted hardware-software debugging directly on the controller, resolving switch activation logic and signal integrity issues.
  • Validated end-to-end telemetry through Raspberry Pi into IGS/OpenMCT, confirming accurate packet reception, parsing, and display.
  • Designed and executed unit tests for the launch controller codebase to harden key functionality under varied simulated conditions.
  • Authored in-depth documentation and led a two-person subgroup, mentoring peers and keeping delivery on track.
