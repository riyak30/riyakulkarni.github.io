Project

AI Developer Tools for Incident Resolution

Microsoft

Software Engineering Intern

Built MCP tools and guidance systems that let AI assistants self-serve incident triage, navigate hierarchical data, and generate boilerplate, cutting manual developer effort from minutes to seconds.

MCP tools C# Node.js Developer productivity AI-assisted workflows

The Problem

Two separate but related inefficiencies were slowing the team down. First, GitHub Copilot was nearly unusable in our codebase — the repo was so large that the AI would spend up to 20 minutes just orienting itself before producing hallucinated or incorrect implementations, leaving developers to spend more time cleaning up after it than they saved. Second, engineers resolving "Shield" incidents (user-reported Outlook issues) had to manually download incident files, navigate across multiple apps and data sources, and manually piece together context just to arrive at a solution — often an incomplete one given the fragmented information available.

What I Did

  • Automated customer incident resolution by developing 2 specialized MCP (Model Context Protocol) tools that enable AI assistants to autonomously navigate hierarchical data structures and extract relevant information from user reports, saving 10-20 minutes per incident and eliminating need for manual developer intervention in support workflows.
  • Accelerated development processes from 30 minutes to 5 seconds by implementing 15+ automated boilerplate code generation MCP tools for core software components, while ensuring consistent coding patterns and standards across enterprise codebase.
  • Eliminated AI processing bottlenecks by developing 3 MCP investigation tools that enable AI agents to efficiently parse complex object definitions without reading 10,000+ line files, reducing analysis time from minutes to seconds.
  • Built context-aware AI guidance system using 9 specialized MCP instruction guides that automatically activate based on task context, enabling AI assistants to provide expert-level guidance for complex software engineering workflows without manual knowledge lookup.

The Outcome

AI assistants could navigate the codebase in seconds rather than minutes, and incident triage no longer required engineers to manually gather context across multiple tools. Shield incident resolution went from a fragmented, multi-step process to an AI-assisted workflow, saving 10–20 minutes per incident. Boilerplate generation that previously took ~30 minutes was reduced to 5 seconds, and the guidance system meant AI had the right context automatically — no manual knowledge lookup required.
